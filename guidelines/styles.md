# PrimeReact Style Guidelines

## Layout
- Use Tailwind flex/grid for layout
- Default container: `max-w-6xl mx-auto`
- Page padding: `p-6`

## Typography
| Role | Class |
|---|---|
| Page title | `text-3xl font-bold` |
| Section title | `text-xl font-semibold` |
| Card title | `text-lg font-medium` |
| Body | `text-base` |
| Caption | `text-sm text-[var(--text-color-secondary)]` |

## Border Radius
- Cards: `rounded-lg`
- Custom badges: `rounded-full`

## Shadows
- Subtle: `shadow-sm`
- Elevated: `shadow-md`
